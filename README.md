## Browser
Browser - brave (suggested, if not make sure you have anti canvas)
- ClearURLS - [github.com/ClearURLs/Addon](https://github.com/ClearURLs/Addon) (Needs to be sideloaded)
- Ublock origin -[github.com/gorhill/uBlock](https://github.com/gorhill/uBlock?tab=readme-ov-file#ublock-origin-ubo)
- No script - [noscript.net](https://noscript.net/)
## Operating System
- user friendly (fedora KDE) - [fedoraproject.org/kde/](https://fedoraproject.org/kde/)
- user friendly, gaming - [nobaraproject.org](https://nobaraproject.org/)
- Extremely private, not a daily driver (TAILS) - [tails.net](https://tails.net)
